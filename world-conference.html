CMAA Events

Challenge

I currently work as the sole Web Designer and Front-End Developer at Club Managers Association of America (CMAA). This organization educates and certifies hospitality professionals who manage private clubs through an series of conferences that each have their own focus. Prior to my role at CMAA each event was represented by a set of pages within the main association site. This caused confusion for attendees and hindered the messaging. Though each conference was branded with a logo and color scheme, these distinctions were lost in the vast network of pages. Additionally, the main CMAA site lacked a responsive framework which made content less accessible on mobile devices.

Approach

I saw an opportunity to make use of the existing event assets by expanding them into fully realized microsites where attendees could access all pertinent information to an event. This eliminated confusion and over-crowding of content and expressed the experiential qualities of each event.

This project also afforded me the flexibility to introduce elements of a unifying visual language across all CMAA products. I developed and introduced a set of fully responsive components to accommodate use across all devices and set standards for how users engage with and navigate through content.
